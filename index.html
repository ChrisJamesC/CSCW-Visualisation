CSCW Visualization

This page represents the projects of Sergio Almaguer, Christopher Chiche Thomas Droxler and Miji Kim in the EPFL class Computer Supported Cooperative work. For this project 3 different visualizations were made, using some different datasets.

These visualizations were made using the d3.js and nvd3.js libraries.
